## Operator Logika
Operator digunakan untuk menentukan logika dari operasi yang nanti kita buat

| Operasi | Description |
| ----------- | ----------- |
| && | And |
| (pipe)(pipe) | Or |
| ! | Not |

Cara penggunaannya contoh :
```
x =10
10
(x % 2 == 0)
true
(x % 2 == 0) && (x < 10)
false
x = 9
9
(x % 2 == 1) && (x < 10)
true
```
Untuk penggunaannya coba ingat ingat sistem digital.
